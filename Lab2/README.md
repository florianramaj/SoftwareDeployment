# Dokumentation wie die Releases freigegeben werden

Unter Releases -> Create a new release -> Create -> dann warten bis der Stage erledigt ist, wenn der Stage fertig ist dann auf Aprove klicken (siehe Lab2/Release_Approve.PNG), dann kann der ausgewählte User durch ein Klick auf "Approve" bestätigen (siehe Lab2/Release_Approve2.PNG); somit startet dann der nächste Stage.
